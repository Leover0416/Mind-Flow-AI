# MINDFLOW

MINDFLOW 是一个面向 AI Agent 场景的全栈项目，目标是把「大模型 + 工具调用 + 知识检索 + 实时反馈」整合成可落地的应用系统。

## 项目定位

这个项目不只是一个聊天页面，而是一个可扩展的 Agent 平台，支持任务规划、工具执行和知识问答等能力。  
适合作为 AI 工程化实践项目、个人作品集项目和面试展示项目。

## 核心能力

- Agent 循环执行（思考 -> 决策 -> 执行 -> 反馈）
- 工具调用框架（支持扩展自定义工具）
- RAG 检索增强（向量检索 + 知识库问答）
- 多模型接入与切换
- SSE 实时推送执行过程

## 技术栈

- 后端：Java、Spring Boot、Spring AI
- 前端：现代 Web 前端（位于 `ui` 目录）
- 数据层：PostgreSQL + 向量检索能力
- 通信：REST API + SSE

## 项目结构

- `jchatmind`：后端服务与 Agent 核心逻辑
- `ui`：前端界面与交互
- `examples`：示例数据或演示资源

## 快速开始

1. 克隆仓库
2. 配置数据库与模型 API Key
3. 启动后端服务
4. 启动前端项目
5. 打开浏览器进行体验

> 提示：建议在本地准备 `.env` 或配置文件，统一管理密钥与连接信息。
